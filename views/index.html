Request Header Parser Microservice

Description

This service provides the IP address, preferred languages and system info for your device.

Simply add /api/whoami to the tail-end of this url!

Example API Call

https://handsome-xylophone.glitch.me/api/whoami

Example Output:

{"ipaddress":"::ffff:159.20.14.100",
"language":"en-US,en;q=0.5",
"software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
